Relaciones Molares

Masa del elemento → Número de moles

Número de moles = Masa del elemento / Masa molar

Número de moles → Número de átomos

Número de átomos = Número de moles × Número de Avogadro

Masa del elemento → Número de átomos

Número de átomos = (Masa del elemento / Masa molar) × Número de Avogadro

Número de moles → Masa del elemento

Masa del elemento = Número de moles × Masa molar
